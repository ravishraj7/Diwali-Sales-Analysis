# Diwali Sales Analysis

## Project Overview
This project analyzes Diwali sales data to understand customer purchasing behavior and identify business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Most buyers belong to the 26-35 age group.
- Female customers contributed more sales than male customers.
- IT, Healthcare, and Aviation professionals spent the most.
- Food, Clothing & Apparel, and Electronics categories generated the highest sales.
